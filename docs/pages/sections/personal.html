👤 个人中心 — 个人资料与学习统计

个人中心是用户在阿米加的 数字身份主页，集中展示个人资料、学习统计数据、成就徽章，并提供资料编辑入口。

设计目标： 给每个学习者一个"学习简历"——不仅展示"我是谁"，更重要的是展示"我学到了什么"和"我达到了什么水平"。

资料展示

  • 头像： 默认使用 😊 中性表情符号，支持上传自定义头像
  • 昵称： 显示用户名，支持修改（每月限 1 次）
  • 学习语言： 显示用户正在学习的语言对及当前 CEFR 级别
  • 学习天数： 显示连续学习天数（Streak）和累计学习天数
  • 简介： 简短的自我介绍，可选填

学习统计

指标说明展示方式
总 XP累计获得的总经验值大数字 + 进度条（下一目标）
已完成技能数所有级别中已完成的技能节点总数数字 + 占全部技能的比例
最高 Streak历史最长连续学习天数徽章式展示
AI 口语对话次数完成的 AI 口语练习次数数字 + 周趋势折线图
单词掌握量单词本中标记为"已掌握"的词汇数数字 + 占目标词汇量的比例
周学习时长本周累计学习分钟数进度环 + 与上周对比的百分比

成就徽章系统

徽章分为三类：

  • 学习里程碑： 完成 10 个技能、100 个技能、达到 B1 级别、达到 C1 级别等
  • 连续学习： 7 天、30 天、100 天、365 天连续学习
  • 社区贡献： 回答 50 个问题、帮助纠错 100 次、获得 100 个赞等

未解锁的徽章以灰色轮廓显示，点击可查看解锁条件（"再完成 3 个技能即可解锁"）。

个人资料编辑

点击编辑按钮进入编辑模式：

  • 修改头像（系统提供多组默认 emoji + 自定义上传）
  • 修改昵称（每月限 1 次，显示剩余次数）
  • 修改简介
  • 设置所在城市和国家（影响排行榜地域维度）
  • 学习目标设置（每日学习时长目标、每周 XP 目标）
💡 个人中心不仅仅是"我的页面"，更是向外展示学习成果的窗口——用户可以将个人主页链接分享给朋友，朋友无需登录即可查看其学习进度和成就徽章。

⚙️ 设置 — 应用偏好与账户管理

设置模块提供 阿米加应用的 全局配置管理，涵盖学习偏好、语言切换、通知管理、账户与隐私设置等。

设计目标： 让每个学习者都能根据自己的习惯和需求定制学习体验，同时确保账户安全和隐私可控。

⚙️ 设置

目前仅提供 界面语言 设置，其他设置项将在后续版本中添加。

设计目标： 保持设置页精简，按需逐步扩展。核心学习偏好（目标语言、每日时长等）在新用户向导中完成设置，并可在个人中心的"学习目标"入口调整。

🌍 界面语言

切换应用的界面显示语言，不影响学习内容的目标语言。

  • 中文 — 默认界面语言
  • English — 英文界面
  • Español — 西班牙语界面

界面语言独立于学习语言。例如：你可以使用中文界面学习西班牙语，或使用西班牙语界面学习中文。

🤖 大模型 API 设置

阿米加的新闻阅读 AI 改写、口语练习等智能功能均依赖大语言模型 API。用户需自行配置 OpenAI 兼容的第三方 API（如 OpenAI、DeepSeek、智谱、硅基流动等），应用不提供内置模型服务。

设计目标： 提供主模型和备用模型两套配置，主模型不可用时自动切换到备用模型，确保 AI 功能的高可用性。所有配置仅存储在本地 SQLite，不上传至任何服务器。

交互说明

  • 主模型和备用模型分开展示为两个可折叠的卡片区域，主模型卡片默认展开
  • 每个配置区域右上角有「测试连接」按钮，点击后调用 /v1/models 接口验证 API 可用性
  • 测试结果以 Toast 显示：✅ "连接成功，模型可用" / ❌ "连接失败，请检查配置"
  • API Key 输入框默认掩码显示，点击眼睛图标切换明文/密文
  • 保存配置时自动运行一次连接测试，失败则弹出确认对话框："API 连接测试失败，是否仍要保存？"
  • 首次进入应用时，若未配置任何 API，在触发 AI 功能时弹出引导对话框，引导用户填写主模型配置

降级策略

  • 主模型请求失败（网络错误 / 超时 / 4xx / 5xx）时，自动切换到备用模型，并在页面顶部显示橙色提示"已切换到备用模型"
  • 备用模型也失败时，显示错误提示"AI 功能暂时不可用，请检查 API 配置"，并提供"重新配置"按钮
  • 切换过程中用户操作不中断，当前请求自动用备用模型重新发起
💡 API Key 仅存储在本地 SQLite 数据库中，不会上传到任何服务器。建议使用专用 API Key 而非主账号密钥，并定期轮换。更多设置项（通知管理、隐私控制、主题切换等）将在后续迭代中逐步添加。
