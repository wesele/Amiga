📱 导航与 UI 结构

采用底部 4 Tab 导航体系结构，覆盖学习、排行、互动、个人四大核心领域。练习页面和 AI 对话采用独立全屏模式，新用户首次启动时先走全屏向导流程（三步注册+设置）再进入主界面。默认首页为学习功能枢纽页面。

🏠

学习

默认首页，为功能枢纽（Hub）页面，展示三大核心入口：🏆 晋级之路（CEFR 分级 A1→C1 树状地图）、📖 单词本（词汇分组/闪卡/听写）、🤖 AI 口语练习（自由对话/场景扮演）。顶部显示每日 XP 目标和连胜天数。

🏆

排行

排行枢纽页面，上方为个人排名摘要卡片（当前排名、本周 XP、距上一名差距），下方提供两个子功能入口：排行榜（好友/全球/城市三级排名 + 段位系统青铜→钻石）和世界地理（全球学习者分布地图）。每周一结算并发放段位奖励。

👥

互动

社交枢纽页面，采用双屏流程：好友列表 → 聊天窗口。支持双语聊天（底部弹出方向切换面板：中文→外语 / 外语→中文 / 双语对照）、好友搜索与添加、实时 PK 对战邀请。语言互助配对功能已合并至社交聊天场景中。

👤

个人

用户档案页面，封面头像 + 姓名 + 当前学习语言/等级头栏，下方为菜单列表：个人资料（头像/昵称/简介）、学习语言（已学语言对及等级）、账号绑定（微信/Google）、设置。成就徽章、学习统计、目标管理等分散在各功能模块中管理。

UI 设计原则

📱

移动优先

以 480×800 为基准设计尺寸，所有交互元素最小触摸区域 ≥ 44px。桌面端保持相同布局但增加留白，不引入额外的桌面专用视图。

⚡

即时反馈

每次答题、切换 Tab、点击按钮都有 100–200ms 内的视觉/动效反馈。正确/错误/加载/成功四种状态各有专属配色和微动画，降低用户等待焦虑。

🎯

内容优先

学习场景（练习页、晋级之路）占据全屏，底部导航在练习和 AI 对话时自动隐藏。所有 UI 元素都为学习内容让路，减少不必要的装饰性干扰。

🧭

明确导航

底部 Tab 始终显示当前所在页面，Tab 切换有平滑过渡动画。页面标题和返回路径清晰可见，用户在任何深度页面都知道如何返回上级。

♿

无障碍

所有可交互元素支持屏幕阅读器标签，色彩对比度符合 WCAG AA 标准（主色/背景对比度 ≥ 4.5:1），支持系统字体缩放和减少动效模式。

🌙

深色模式

内置浅色/深色两套主题，跟随系统设置自动切换。深色模式下所有颜色的亮度降低但不饱和，保持可读性和品牌识别度。
