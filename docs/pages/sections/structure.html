📚 内容结构

采用五层内容层级体系，从宏观到微观逐级细化。线性主线搭配分支关卡，覆盖 CEFR A1 到 C1 全阶段，每级设有明确的解锁条件，确保学习节奏科学可控。

五层内容层级详解

🌐

语言 Language

顶层容器，代表一个完整的学习语言对（如「中文→英语」）。每个语言对包含独立的课程树、词汇库和语法规则表，拥有专属的课程编排和难度曲线。

📚

课程 Course

按主题划分的学习阶段，如「基础入门」「旅行用语」「商务沟通」。每个课程对应一个 CEFR 级别区间，包含 6–12 个单元，完成后解锁下一课程。

📖

单元 Unit

围绕特定语法点或场景主题的练习集合，如「现在完成时」「餐厅点餐」「问路指路」。每个单元包含 4–8 个关卡，以渐进难度编排。

🎯

关卡 Skill

最小的可解锁内容块，包含一组围绕同一知识点的练习序列（5–10 道题）。关卡类型包括主线关卡、分支关卡（语法专项、词汇专项）和挑战关卡。

✍️

练习 Exercise

最小的学习单元，即一道题目。支持多种题型：选择题、拼写题、配对题、翻译题、口语跟读、听力填空等。每道练习附有正确反馈和错题解析。

课程路径特点

📏

线性主线

课程从入门到精通呈线性路径，用户按顺序依次学习课程和单元。每个主线节点完成后自动解锁下一节点，确保知识体系的渐进性和连贯性。

🔀

分支关卡

在线性主线之外设置专题支线节点，如「动词变位专项」「高频词汇 200」「发音纠音课」。分支关卡可选修，完成后给予额外 XP 和成就奖励。

🔓

解锁条件

每个关卡解锁需同时满足：前置关卡全部完成（⭐ 评价）、当日已获得足够 XP 值、无待复习的旧知识点。关键节点需达到特定 CEFR 阶段方可解锁。

📊

CEFR 映射

课程体系完整映射欧洲共同语言参考标准。A1（约 50 课时）→ A2（约 60 课时）→ B1（约 80 课时）→ B2（约 100 课时）→ C1（约 120 课时），每阶段结束时设有水平测试。

🔄

间隔复习

基于遗忘曲线算法，已通过的关卡会按 1天→3天→7天→30天 的间隔自动标记为「需复习」。复习关卡出现在路径地图上，完成复习可巩固知识并获得额外 XP。

🔄

跳级测试

允许已有基础的学习者通过前置测试直接跳至更高课程。测试包含该课程全部知识点的代表性题目，答对率 ≥ 80% 即可跳级，避免重复学习已知内容。
