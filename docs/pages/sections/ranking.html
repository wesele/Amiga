🏆 排行榜 — 多维度竞技排名

排行榜系统通过 每周 XP 竞争 激励学习者保持高频学习。支持 全球、城市、国家 三个维度的排名切换，让学习者与身边或全球的学习者良性竞争。

设计目标： 通过透明、公平的排行机制激发学习动力，同时避免过度竞争带来的焦虑感。排行榜强调"与自己比"和"与同城/同国人比"的温和竞争。

排名维度

维度范围适用场景
🌍 全球全平台用户顶级学习者角逐，适合高阶用户激发成就感
🏙️ 城市同一城市的用户本地化竞争，可能发展成为线下学习社群
🇺🇳 国家同一国家的用户中等规模竞争圈，适合日常查看

排名周期与规则

  • 每周一重置： 所有用户的每周 XP 归零，开始新一轮竞争
  • XP 统计范围： 晋级之路练习完成、AI 口语对话、单词本复习、每日签到均可获得 XP
  • 排名分组： 按 CEFR 级别分组排行（A1 组 / A2 组 / … / C1 组），确保同级别学习者竞争
  • 前 3 名奖励： 周冠军获得"本周之星"勋章和双倍宝石奖励

界面布局

排行榜页面顶部为维度切换器（全球/城市/国家）。列表展示前 50 名用户，每行包含：

  • 排名序号（前 3 名用 🥇🥈🥉 图标）
  • 用户头像和昵称
  • 国家/城市标签（带国旗 emoji）
  • 本周 XP 数值
  • 与上一周排名对比的升降指示（↑↓）

好友高亮

已添加好友的条目在列表中特殊高亮（蓝色边框或标签），方便快速查看好友排名位置。

💡 排行榜设计强调"参与感"而非"竞争感"——即使排名靠后，只要在榜上就能看到自己与前列的差距被量化，从而产生"再学一节课就能前进一名"的正向激励。

🗺️ 世界地理 — 学员全球分布热力图

世界地理页面以 交互式地图 展示 阿米加学员的全球分布。用户可以查看不同地区、不同语言的学习者密度，发现世界各地正在学习同一语言的人。

设计目标： 通过视觉化的全球学习社区地图，增强学习者的归属感和全球视野——"你不是一个人在学，全世界有 X 万人正在和你一起学法语/中文/西班牙语"。

地图功能

  • 语言筛选： 按目标语言筛选地图上的学习者分布（如只看学中文的用户分布）
  • 热力图层： 颜色深浅表示该地区学习者的密度，悬停显示具体数据（城市名 + 用户数）
  • 缩放交互： 从全球视图到国家视图再到城市视图的三级下钻
  • 数据统计： 地图侧边栏显示汇总数据（总用户数、最热门的学习语言、增长最快的地区）
  • "我在哪"： 用户自己的位置以特殊标记（🌟）显示在地图上

数据来源与隐私

  • 位置数据来自用户注册时提供的城市/国家信息（非 GPS 实时定位）
  • 用户可以在设置中随时开启/关闭"在地图上显示我的位置"
  • 关闭后统计数据仍然计入总数，但不会显示具体标识
💡 地图页面未来可扩展为"寻找语伴"的入口——点击地图上的城市即可查看该城市的学习者社区、加入本地学习群组。
