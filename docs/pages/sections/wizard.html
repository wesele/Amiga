🎓 新用户向导 v0.1

新用户首次打开应用时的引导流程。通过分步向导收集用户基本信息和学习目标，完成个性化配置后自动跳转到学习首页。无需注册/登录，打开即用。

1️⃣ 基本信息

首次打开应用时，直接进入基本信息收集页面。所有字段均有合理默认值，用户可以一路"下一步"跳过，也可以逐项填写。信息收集的核心目的是为新闻阅读等功能提供个性化基础。

收集字段

字段必填类型默认值用途
昵称是文本输入"学习者"社交展示、排行榜、个人主页
母语是单选列表根据系统语言自动推断决定翻译方向、新闻阅读中的释义语言（例如母语=中文 → 学西班牙语时翻译为中文）
头像否Emoji选择 / 图片上传😊 表情个人主页、排行榜、AI 对话角色展示
国家/地区是下拉搜索选择根据 IP 自动定位新闻阅读的热搜来源地、排行榜地域分组、时区设置
性别否三选一不公开仅用于匿名化数据统计，不对外展示
出生年份否下拉选择—内容年龄适配（未成年模式）；≤13岁自动开启严格隐私保护

字段详细说明

昵称

  • 输入框实时校验：2~20 字符，不支持特殊符号（@# 等），允许中英文混排
  • 若昵称已被占用，输入框下方显示红色提示"该昵称已被使用"并建议可用昵称
  • 右侧显示字符计数（如 "5/20"）

母语

  • 提供常用语言列表（前 10 位：中文、English、Español、日本語、Français、Deutsch、Português、Italiano、한국어、العربية），其余按字母排序
  • 支持搜索过滤：输入拼音或语言名快速定位
  • 根据操作系统语言自动预填（如 zh-CN → 中文，en-US → English）
  • 母语与目标语言不能相同，若冲突则在下一步给出提示

头像

  • 提供 12 个预设 Emoji 头像（😊🤓😎🦊🐱🐶🐻🐼🦉🌟🎯🎨），横滑浏览
  • 支持"拍照"和"从相册选择"两种上传方式（Android 调用相机/相册 Intent，Windows 调用文件选择器）
  • 上传图片自动裁剪为圆形，分辨率不低于 256×256px

国家/地区

  • 下拉列表按大洲分组（亚洲 / 欧洲 / 美洲 / 非洲 / 大洋洲），常用国家置顶
  • 支持搜索：输入国家名或城市名快速定位
  • 自动定位结果在列表顶部高亮显示，用户可手动修改
  • 选择国家后自动设置时区，影响 Streak 重置时间和每日提醒时间
  • 此字段同时作为新闻阅读的热搜来源地（例如选择西班牙 → 推送西班牙热搜新闻）

性别

  • 三个选项按钮：男 / 女 / 不公开，默认选中"不公开"
  • 选择后右侧显示锁定图标，提示"此信息仅用于数据统计，不对外展示"

出生年份

  • 下拉选择器，按年代分组（2010s / 2000s / 1990s / 1980s / 1970s / 1960s 及更早）
  • 选择 ≤13 岁时弹出提示："我们将为你开启青少年模式，部分内容会有所调整"
  • 青少年模式下：社交功能关闭、排行榜匿名显示、新闻内容过滤为适龄内容
隐私优先： 性别、出生年份为非必填项，且默认不公开。用户可在设置中随时修改隐私可见性。未成年用户（≤13岁）自动启用青少年模式。

2️⃣ 学习目标设置

这是向导最关键的一步，决定了用户的学习路径和内容难度。支持多语言并行学习，完成后一键跳转到学习首页。

设置项

设置项选择方式可选值影响范围
目标语言多选 pill 按钮西班牙语 / 中文 / 英语 / 日语 / 法语 / 德语决定学习课程内容
当前水平单选（每语言独立）零基础 / 初级 (A1-A2) / 中级 (B1-B2) / 高级 (C1)决定起始课程级别
学习目标单选日常对话 / 旅行交流 / 考试准备 / 流利掌握影响推荐课程主题和练习密集度
每日时长单选5分钟 / 15分钟 / 30分钟 / 1小时+设定 Streak 目标和学习提醒

默认值策略

每个设置项均有合理的默认值以减少用户决策负担：

  • 目标语言： 根据用户 IP 所在地和操作系统语言推荐。中国用户 → 默认勾选西班牙语；非西班牙语母语者 → 默认勾选西班牙语
  • 当前水平： 默认推荐"零基础"——用户可以在后续练习中自适应调整
  • 学习目标： 默认"日常对话"——覆盖最广泛的学习动机
  • 每日时长： 默认"15 分钟"——门槛最低，保证可持续

自适应调整

向导中设置的等级仅为初始值。用户学习 3~5 课后，系统会根据练习表现自动调整课程难度（升降级），无需手动干预。

词汇库初始化

用户选择"当前水平"后，系统会根据内置的分级词汇表自动初始化用户的词汇状态（user_vocab 表），为新闻阅读的 AI 改写提供基准：

用户选择词汇库初始化
零基础不预加载任何词汇，所有词从阅读中开始积累
初级 (A1-A2)A1 级别约 500 词自动标记为"认识"，AI 改写时不会再作为新词植入
中级 (B1-B2)A1 + A2 级别约 1500 词自动标记为"认识"
高级 (C1)A1 + A2 + B1 级别约 3500 词自动标记为"认识"

初始化设为"认识"而非"掌握"，留有余地让系统在后续阅读中通过"认识/不认识"反馈验证和微调。

3️⃣ 完成 & 跳转

提交所有设置后，系统在后台完成以下操作：

  1. 保存用户资料： 写入本地 SQLite user_profiles 表
  2. 初始化学习路径： 根据选择的语言 + 等级生成初始课程列表
  3. 创建 Streak 记录： 初始化连续学习计数器

完成后播放简短庆祝动画（彩纸效果），显示欢迎信息 + 首日学习建议，然后自动跳转到学习首页（新闻阅读）。整个流程应 在 2 分钟内 完成。

重新向导： 用户可以随时在设置中修改所有向导中设定的项目。首次向导完成后不再自动弹出，但用户可在设置底部的"重新设置向导"入口重新走完全流程。

⚙️ 与设置的关联

学习偏好

向导 Step 2 设置的语言、等级、目标、时长全部映射到设置的"学习偏好"分组中。用户在设置中可随时修改。修改后课程路径也随之调整。

👤

个人资料

向导 Step 1 收集的昵称、头像、性别、年龄对应个人中心的资料编辑区。头像和昵称可随时修改，性别年龄修改后需确认。

可修改
🔄

重新向导

设置底部提供"重新设置向导"入口，点击后清除当前学习进度（保留已获 XP 和成就），重新走完三步流程。适合学习方向发生重大变化的用户。

谨慎操作

⚠️ 边界情况

场景处理方式
用户向导中途退出保存当前步骤状态，下次打开时从断点继续。超过 7 天未完成则重置
用户选择多语言学习为每种选定语言创建独立的 learner_courses，首页显示融合路径（日常可切换焦点语言）
用户从已登录设备重新打开跳过向导直接进入学习首页；未完成向导的用户继续从断点开始
未成年用户（≤13 岁）不收集性别和出生年份精确值，选择年龄段代替；默认关闭所有社交功能；排行榜匿名显示
用户拒绝提供任何个人信息允许跳过 Step 1 全部字段，使用纯默认值。不影响学习功能，仅影响社交展示
选择"零基础"但已有语言基础前 3 课提供快速通道测试，通过后自动跳级，无需走完全部低级课程
母语与目标语言相同在 Step 2 给出友好提示："你的母语已经是 XX 了，是否想学习其他语言？"用户仍可坚持选择
