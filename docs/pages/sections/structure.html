📚 内容结构 — 题型题库设计

晋级之路的核心是一套可复用的题型体系，每种题型对应特定的认知层次和技能训练目标。学习路径由题型组合驱动——不同学习阶段开放不同的题型池，题目内容随 CEFR 级别递进。当前以西班牙语为锚点语言设计，题型与语言解耦，可复用到任何目标语言。

📋 题型总览

共 11 种标准题型，按认知层次分为三个梯队：

第一梯队 · 基础识别（A1 入门期）

目标：建立外语符号与含义的初次连接，以被动识别为主。

ID题型交互形式训练技能认知层次示例
T01图片识词看图片 → 四选一阅读 · 词汇识别🍎 图 → manzana
T02听音选图听发音 → 选对应图听力 · 词汇识别🔊"perro" → 🐕 图
T03双向配对拖拽配对 4 组阅读 · 记忆识别agua ↔ 水；leche ↔ 牛奶

第二梯队 · 理解应用（A1 进阶期 ~ A2）

目标：在语境中理解语言结构，完成半开放式输出。

ID题型交互形式训练技能认知层次示例
T05补全句子选词填空（3-4 选 1）阅读 · 语法理解Yo ___ café. [bebo / bebes / bebe]
T06句子排序拖拽单词排成句语法 · 语序应用[me] [llamo] [Ana] → Me llamo Ana.
T07翻译选择看源语 → 四选一阅读 · 翻译理解"Quiero agua" → 我想要水
T08听力选择听句子 → 三选一听力 · 理解理解🔊"Son las tres" → 选 3:00

第三梯队 · 产出表达（A2）

目标：自主产出语言，从词汇到句子再到简单段落。

ID题型交互形式训练技能认知层次示例
T09拼写输入听音 + 看释义 → 打字拼写 · 听力应用🔊 → 输入 "gracias"
T10翻译输入看源语 → 键盘输入写作 · 翻译应用"我早上喝咖啡" → (Yo) bebo café por la mañana.
T11语音跟读听 → 录音 → AI 评分口语 · 发音应用🔊"Buenos días" → 跟读 → 评分
T12情景回应阅读场景 → 选最佳回应语用 · 交际应用踩到别人脚 → [A] Gracias [B] Lo siento [C] Hola
💡 题型与语言解耦：所有题型定义只描述交互逻辑和认知目标，不绑定具体语言。切换目标语言（如西班牙语→日语）时，题型体系无需改动，仅替换语料内容。

🗺️ 学习阶段与题型映射

每个 CEFR 级别开放特定的题型组合，越往后可用的题型越丰富，从被动识别逐步过渡到主动产出。

CEFR阶段可用题型练习形式示例主题
A1 前半入门识别期T01 · T02 · T03选择为主问候、字母、数字、颜色、食物
A1 后半句型理解期T01–T03、T05–T08选择 + 排序时间、日期、点餐、购物、问路
A2基础表达期T01–T03、T05–T12选择 + 输入 + 录音日常活动、频率副词、爱好、天气、家庭
