🌐 语言支持

采用数据与代码分离的可扩展架构，使平台能无限插拔语言对。起始阶段以 西班牙语为核心锚点语言，集中打磨中文 ↔ 西班牙语双语互学体验，验证学习效果后再逐步扩展至更多语言。

🇨🇳➜🇪🇸

中文 → 西班牙语

面向中文母语者设计，从零基础到流利交流。课程涵盖西班牙语发音（音节与重音规则）、核心语法（名词阴阳性、动词变位系统、常用时态）、日常会话场景以及西班牙及拉美文化背景知识。

首推语言对
🇪🇸➜🇨🇳

西班牙语 → 中文

面向西语母语者学习中文，从拼音和四个声调开始，逐步学习汉字书写、基础语法结构、量词用法及中文特有的话题优先句式，帮助学习者跨越汉字学习障碍。

首推语言对
➕

更多语言对（后续扩展）

所有课程内容以 JSON/YAML 格式定义，与代码逻辑完全分离。添加新语言对只需提供对应语料包和翻译文件，无需修改核心引擎。西班牙语核心体验打磨完成后依次支持英语、日语、韩语、法语、德语、阿拉伯语等热门语言。

可扩展架构

以西班牙语为锚点的设计理念

🎯

深度打磨

与其初期覆盖多语言导致体验平庸，不如将一种语言的课程做到极致。西班牙语作为全球近 5 亿母语者的第二大语言，同时是中文学者最热门的外语选择之一，具备足够的学习者基数来验证产品价值。

🧩

西语特色教学

西班牙语的动词变位体系、虚拟式用法、体貌特征（完成体/未完成体）等都是学习的重点和难点。课程设计将围绕这些特色语法现象进行专项训练，辅以 AI 实时纠错。

🌎

多方言覆盖

同时覆盖西班牙本土西语（castellano）和拉美西语（español latinoamericano）的主要差异，包括 vosotros 的使用、seseo/ceceo 发音区别、以及各地词汇差异（如 computadora vs. ordenador）。

📂

数据驱动

课程数据、词汇表、语法规则、练习模板均以结构化数据文件存储。每个语言对独立维护一套数据目录，包含课程编排、例句、音频映射和本地化字符串。

⚙️

通用引擎

练习引擎、语音评测、进度追踪等核心模块与具体语言无关。同一套答题交互逻辑适用于任意语言对，仅需替换题目数据和语音模型。

🧩

插件式添加

新语言对通过注册 manifest.json 声明语言信息、字体资源、TTS 语音标识即可接入。支持社区贡献者通过 Pull Request 提交语言包。

🗺️

未来路线

首批验证中文↔西班牙语之后，计划依次接入英语、日语、韩语、法语、德语、阿拉伯语、俄语、葡萄牙语、意大利语、泰语、越南语等，覆盖亚洲、欧洲主要语言。
