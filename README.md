## 🇨🇳 Amiga - 跨平台 AI 外语学习应用 (中文)

### ✨ 核心功能亮点

- **📰 AI 沉浸式双语新闻阅读**：实时生成并翻译中英/中西/多语种平行新闻资讯，CEFR 难度自动分级，支持长难句 AI 句法剖析。
- **💬 SoulMate 灵伴故事与情景口语**：自定义 AI 伙伴性格与剧情场景（职场面试、旅行穿梭、日常唠嗑），支持语音对话与精准发音指导。
- **📺 Android TV 10-foot 遥控大屏体验**：专为电视客厅大屏重构的视觉与交互，全遥控器按键适配，支持上下左右 D-Pad 方向键无缝切焦。
- **🧠 智能词库与 SuperMemo SRS 记忆算法**：根据遗忘曲线自动调度复习节奏，涵盖拼写、单选、听音选词等多种练习。

## 🇬🇧 Amiga - Cross-Platform AI Language Learning App (English)

**Amiga** is a 100% free and open-source AI-powered foreign language learning companion built with **Tauri 2.x (Rust) + Vue 3**. Designed for Mobile, Android TV, and Windows Desktop.

Combining LLM intelligence with immersive language techniques, Amiga offers parallel AI news reading, interactive roleplay conversation stories (SoulMate), SRS vocabulary revision, and a dedicated 10-foot Android TV experience.

### ✨ Highlights & Features

- **📰 AI Parallel News**: Daily multi-lingual news with CEFR level tags (A1-C2), tap-to-translate tooltips, and grammar tree breakdown.
- **💬 SoulMate AI Companion**: Interactive roleplay storylines, live voice chat, speech-to-text, and adaptive response suggestions.
- **📺 Android TV 10-foot Interface**: Optimized remote D-Pad navigation designed for living room big screens.
- **🧠 Smart SRS Memory System**: SuperMemo spaced repetition algorithm with flashcards, spelling, and listening quizzes.

## 🛠️ Local Development & Quick Start

```bash
# Clone the repository
git clone https://github.com/wesele/Amiga.git
cd Amiga

# Install dependencies
npm install

# Run Web Dev Server
npm run dev

# Run Windows Desktop App (Tauri)
run-windows.bat

# Run Tests
npm run test:all
```
